📘 Aula de Python – Fundamentos da Linguagem

Este repositório contém materiais introdutórios para o aprendizado da linguagem Python, abordando desde conceitos básicos até exemplos práticos para fixação.

Durante a aula, são explorados temas como:

Sintaxe básica e estrutura do Python
Tipos de dados (inteiros, strings, listas, dicionários)
Estruturas de controle (if/else, loops)
Funções e organização de código
Boas práticas de programação

O conteúdo foi desenvolvido com foco em iniciantes, utilizando exemplos simples e didáticos para facilitar a compreensão.

🚀 Objetivo:
Proporcionar uma base sólida em Python para que o aluno consiga desenvolver seus primeiros programas e evoluir para projetos mais avançados.

📂 Estrutura do repositório:
Os arquivos estão organizados por tópicos, permitindo acompanhar a evolução do aprendizado de forma progressiva.
